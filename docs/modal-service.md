# Modal Service

## Overview

Provides a centralized service for opening and managing application modals.

```text
Component
  ↓
Modal Service
  ↓
Modal Store
  ↓
Modal Provider
```

## Responsibilities

- Open and close modals
- Pass modal data
- Manage modal state
- Support reusable dialogs

## Benefits

- Consistent modal behavior
- Reusable components
- Cleaner UI logic
